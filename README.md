#  Profile Setup Application

A multi-step profile setup form built using React, TypeScript, Redux Toolkit, Material UI, and React Hook Form.

---

##  Tech Stack

- **Vite**
- **React**
- **TypeScript**
- **Redux Toolkit**
- **React Hook Form**
- **Material UI**
- **Day.js**
- **React Router**

---
